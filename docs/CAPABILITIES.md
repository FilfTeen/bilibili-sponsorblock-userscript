# 能力说明

## 1. SponsorBlock 片段能力

脚本会基于视频 `bvid` 请求 SponsorBlock 片段，并按分类与动作类型执行：

- `skip`
- `mute`
- `poi_highlight`
- `full`

每个分类支持：

- `自动`
- `手动`
- `仅提示`
- `关闭`

## 2. 整视频标签能力

整视频标签会综合以下来源：

- SponsorBlock 社区 `full` 标签
- 整视频标签接口
- 本地评论/页面线索
- 用户本地保留/忽略记录

显示位置包括：

- 视频标题前胶囊
- 首页/搜索/历史/推荐卡片胶囊

## 3. 评论区增强

- 评论广告/带货识别
- 可疑广告回复识别
- 评论属地显示
- 命中评论折叠与恢复

## 4. 动态页增强

- 可疑商业动态识别
- 根据配置进行标记或折叠

## 5. 本地知识库与自学习

脚本会维护本地辅助知识：

- 手动保留本地标签
- 忽略某个视频的本地判断
- 本地商业意图知识库

这类能力的目标是“增强本地使用体验”，不是制造伪精确结论。

## 6. 低侵入 UI

脚本尽量通过独立节点和局部挂载工作，而不是大面积改写原生页面：

- 标题胶囊
- 缩略图胶囊
- 紧凑视频顶栏
- 设置 / 帮助控制台
- 低打扰通知浮窗

## 7. 不提供的能力

- 片段投稿
- 完整投票/账号流程
- 独立扩展后台
- 完整的扩展 popup/options 页面

这些能力要么需要浏览器扩展后台，要么需要更高侵入性的站内接管，当前 userscript 不提供。
