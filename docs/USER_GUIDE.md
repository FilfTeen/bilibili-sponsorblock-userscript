# 使用手册

## 适用对象

BSC 面向在 Safari + Tampermonkey 中使用 Bilibili 的用户。它提供广告片段辅助跳过、整视频商业性质提示、评论/动态广告辅助识别、属地显示和本地学习能力。

## 安装与入口

1. 安装 Tampermonkey。
2. 导入 `dist/bilibili-sponsorblock.user.js` 或 GitHub raw 安装链接。
3. 打开支持的 Bilibili 页面。
4. 从 Tampermonkey 菜单点击 `打开 BSB 控制台`，或在视频页点击标题胶囊/播放器盾牌按钮进入控制台。

Tampermonkey 菜单只保留一个控制台入口，避免“打开”和“切换”同时出现造成混淆。

## 推荐初始配置

- 保持 `启用 Bilibili SponsorBlock` 开启。
- 评论区过滤先用 `仅标记，不隐藏`。
- 动态过滤先用 `仅标记，不隐藏`。
- 评论区属地显示保持开启。
- 本地判断以“辅助提示”为准，不要在没有观察样本前激进隐藏内容。

## 视频页

视频页包含：

- SponsorBlock 片段跳过。
- 播放器进度条片段标记。
- 标题前整视频标签。
- 右侧推荐卡片缩略图标签。
- 紧凑视频顶部栏。

### 标题胶囊

标题前出现胶囊时：

- 点击胶囊可查看分类说明。
- 上游标签可提交“标记正确 / 标记有误”。
- 本地标签可选择保留或忽略。
- 如果上游已有整视频标签，本地推理和评论反馈不会覆盖它。

### 跳过提示

当脚本检测到片段时，会按分类配置自动处理或提示：

- `撤销`：回到刚跳过片段的开始处。
- `保留本段`：本次保留当前片段，不再重复抢跳。
- `立即跳过`：手动跳到片段结束。

### 紧凑视频顶部栏

紧凑顶部栏用于在视频页提供搜索和账号入口：

- 默认开启。
- 在网页全屏、原生全屏和播放器全屏状态下隐藏。
- 可选择是否显示 B 站原生搜索框中的灰字广告文案。
- 可选择是否允许空输入时搜索当前可见灰字文案。

## 评论区

评论区增强包含：

- 商品卡广告识别。
- 导流、优惠、购买闭环等文本识别。
- 可疑托评识别。
- 评论属地显示。
- 命中评论标记或折叠。
- 本地视频标签反馈入口。

过滤模式：

- `关闭`：不处理评论广告识别。
- `仅标记，不隐藏`：显示标签，但不折叠内容。
- `隐藏内容并保留提示`：折叠命中评论，并保留恢复入口。

评论属地只显示 Bilibili 当前 payload 已有的字段，不推断真实位置。

## 动态页

动态页增强用于识别明显商业动态：

- 商品卡动态。
- 明确促销、领券、购买、导流内容。
- 可疑商业转发。

普通活动、资讯、玩梗、引用、反讽和弱商业词会保守处理。建议先使用 `仅标记，不隐藏`。

## 标签配色与透明度

控制台可调整：

- 分类标签颜色。
- 评论广告标签颜色。
- 评论属地标签颜色。
- 标题、缩略图、评论、动态标签透明效果。

颜色编辑器采用“草稿预览 + 显式应用”：

- 拖动颜色只更新预览，不立即保存。
- 点击 `应用` 或按 Enter 才保存。
- 出现未保存改动时才显示取消操作。

## MBGA

MBGA 默认开启，用于减少部分生态噪音：

- 阻断部分追踪/遥测请求。
- 压制 PCDN/WebRTC 相关能力。
- 清理追踪参数。
- 动态页宽屏。
- 视频裁切模式入口。
- 页面灰度和复制限制修正。

如果出现播放、加载或页面兼容异常，可以先在控制台关闭 MBGA 或单独关闭相关子项。

## 维护工具

控制台概览页提供：

- `清理缓存`：清除 SponsorBlock 和整视频标签缓存，不删除配置。
- `恢复默认设置`：恢复脚本配置。

恢复默认设置有二阶段确认。点击后需要在短时间内再次确认，防止误触。

## 可靠性提醒

可以较放心参考：

- SponsorBlock 已收录片段。
- SponsorBlock 已收录 `full` 标签。
- Bilibili payload 自带属地字段。

需要自行甄别：

- 本地推理得出的整视频标签。
- 评论/动态启发式广告识别。
- 评论反馈引发的本地学习结果。

详细边界见 [RELIABILITY.md](./RELIABILITY.md)。
