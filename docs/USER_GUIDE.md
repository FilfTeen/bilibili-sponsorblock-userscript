# 使用手册

## 目标用户

这个脚本面向已经在 Safari 或其他支持 Tampermonkey 的浏览器里使用 Bilibili 的用户，目标是提供：

- 视频广告片段的辅助跳过
- 整视频商业性质提示
- 评论区和动态页的广告辅助识别
- 评论区属地显示
- 面向本地使用习惯的控制台和配置项

## 安装

1. 安装 Tampermonkey。
2. 导入 `dist/bilibili-sponsorblock.user.js`。
3. 打开 `https://www.bilibili.com/` 或任意视频页。
4. 通过视频页顶部紧凑栏、标题胶囊或 Tampermonkey 菜单进入控制台。

## 推荐使用方式

### 初次使用

推荐先这样配置：

- 保持 `启用 Bilibili SponsorBlock` 为开启
- 评论区过滤先用 `仅标记，不隐藏`
- 动态过滤先用 `仅标记，不隐藏`
- 评论区属地显示保持开启
- 若你更在意误判，先不要把本地增强开得太激进

### 视频页

视频页主要有 4 类能力：

- 标题前整视频标签
- SponsorBlock 片段自动/手动跳过
- 播放器进度条片段标记
- 播放页右侧推荐卡片标签

当标题前出现标签时：

- 点击标签可以查看说明
- 如果该标签来自可反馈的数据源，可以提交“标记正确 / 标记有误”
- 如果标签来自本地线索，会提示这是本地辅助判断，不会伪装成社区共识

### 跳过提示浮窗

当脚本检测到你主动跳进广告片段时，会显示一个低打扰浮窗：

- `保留本段`：本次保留这个片段，直到离开该片段前不再自动抢跳
- `立即跳过`：立刻跳到片段结束

如果是脚本刚执行过自动跳过，你可以用 `撤销` 回到片段开始处。

### 评论区

评论区目前有两组增强：

- 评论区广告/带货识别
- 评论属地显示

如果评论被命中：

- `仅标记` 模式下只显示标签
- `隐藏并保留提示` 模式下会折叠内容，并提供恢复入口

### 动态页

动态页增强只用于识别明显商业内容，不会尝试替代视频 SponsorBlock 数据。

## 主要配置说明

### 片段与标签

- `启用 Bilibili SponsorBlock`
  关闭后，视频片段请求、标题标签、缩略图标签和播放器增强全部停用
- `启用缓存`
  用于缓存 SponsorBlock 和整视频标签结果，减少重复请求
- `显示播放器进度条标签`
  在视频进度条上显示分类片段
- `首页 / 列表卡片标签`
  控制首页、搜索页、历史页、推荐区等缩略图标签的显示方式
- `最短处理时长`
  比这个更短的片段默认不处理，用于降低误触发

### 动态 / 评论

- `动态过滤模式`
  控制动态页商业内容的处理方式
- `评论过滤模式`
  控制评论区广告内容的处理方式
- `同时隐藏命中评论的回复`
  主评论命中后，是否联动折叠对应楼层回复
- `评论区属地显示`
  默认开启，只在页面实际返回属地字段时显示

## 什么时候应该相信脚本，什么时候不应该

可以优先参考脚本的情况：

- 社区已收录的 SponsorBlock 片段
- 社区已收录的整视频 `full` 标签
- 评论 payload 自带的属地字段

需要你自行甄别的情况：

- 本地规则判断出的商单/导流标签
- 评论区和动态页的广告启发式识别
- 社区未收录、但页面线索明显的商业视频
